加载模块文件的内容

程序集：  ModuleContract (在 ModuleContract.dll 中) 版本：0.1.0.1 (0.1.0.1)
语法
C#
复制
IViewDoc LoadFile(
	byte[] content,
	int version
)

参数

content
类型：SystemByte
模块文件的二进制序列化内容
version
类型：SystemInt32
主文件的版本，每个ET模块的实现中需保证模块文件内容向后兼容

返回值

类型：IViewDoc
返回加载内容后的IViewDoc接口
