ET.ModuleInterface 命名空间

  • 本命名空间下的成员是ET模块的抽象接口定义以及相关辅助类实现（释：ET模块是ET设计器的功能组件）
  • 本命名空间仅在ModuleInterface程序集中进行定义，空间内成员被ET主程序和ET模块各自独立引用
类
  类说明
ETEventArgs
ET路由事件使用的事件参数类
ETPage

ET页面类

ET页面继承自WPF的布局控件Syste.Winows.Controls.Grid，并加入ET自定义路由事件

ET页面是ET模块中UI接口的统一包装器

ModuleHeaderAttribute
ET模块的元数据信息，延迟加载时主程序只读取该属性信息
接口
  接口说明
ICommModule

ET模块的通用接口定义，所有的ET模块均需要实现此接口

ET设计器的每个功能均由独立的ET模块实现，ET模块内部采用MVVM模型实现其功能

IViewDoc

模块文件控制器接口定义

IViewDoc接口对应ET模块内MVVM模型的VM部分，通常为已加载模块文件内容的可视化组件
